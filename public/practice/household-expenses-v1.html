Business Context (Summary)

This page represents a household expense tracking system using the /api/v1/data/random/costs endpoint.
The application provides:

  • Daily transaction details with categories
  • Running balance calculations
  • Transaction count statistics

Core Features

  • Data Display:
    • Transaction date and category details
    • Category icons (emojis) and names
    • Individual transaction amounts
    • Visual chart (7-day view only)
  • Calculations:
    • Daily Balance: Sum of day's transactions
    • Running Balance: Cumulative sum from current day forward
    • Transaction Count: Number of entries per day
  • Risks: The frontend logic is critical for accurate financial tracking and reporting

Important notes

  • All amounts are formatted in Polish currency (PLN)
  • Days with no transactions display "No transactions"
  • Running balance includes current and future days
  • Missing or invalid amounts may affect calculations

Known Limitations

  • Chart visualization only in 7-day view
  • Random data changes on each refresh
  • Predefined transaction categories
  • Edge cases with undefined values may cause formatting issues
